AERROR_TYPES :string

Errors for Apollo. All errors generated from Apollo which are not directly fired by Cassandra should belong to this list.
Errors have a property name which is always ERR_NAME_PREFIX.AERROR_TYPES, for example apollo.model.save.unsetkey
You can rely on this property to check the error in callback and in try/catch.
